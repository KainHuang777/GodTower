# 遊戲音訊資產目錄 (Audio Assets Directory)

本目錄用於存放「五行迷宮塔防」遊戲所需的背景音樂 (BGM) 與音效 (SFX) 檔案。

## 預設音訊檔案命名規範

為便於前端代碼載入與播放，建議將您的音樂/音效檔案依照以下名稱放置於此目錄：

### 1. 背景音樂 (BGM)
- **主戰鬥背景音樂**: `bgm.mp3` 或 `bgm.wav` (建議使用 OGG 或 MP3 格式以獲得最佳瀏覽器相容性)

### 2. 遊戲音效 (SFX)
- **防禦塔射擊音效**: `shoot.mp3`
- **怪物死亡音效**: `enemy_death.mp3`
- **合成成功音效**: `merge_success.mp3`
- **UI 點擊音效**: `click.mp3`
