# 五行迷宮塔防 (Wuxing Maze TD) - 遊戲設計文件 (GDD)

---

## 1. 遊戲概覽 (Game Overview)
*   **遊戲名稱**：五行迷宮塔防 (Wuxing Maze Tower Defense)
*   **遊戲類型**：迷宮塔防 / 動態尋路策略 / 天賦養成
*   **目標平台**：Web (PC 瀏覽器)
*   **核心玩法**：
    *   地圖上設有起點、終點（基地）以及 5 個必須按順序訪問的檢查點。
    *   玩家在 80x40 網格上擺放五行（金木水火土）+ 陰陽屬性的砲台。
    *   砲台同時作為「牆壁」阻擋怪物尋路，迫使怪物繞行最長路徑。
    *   **路線預覽功能**：戰鬥中隨時點擊「路線預覽」按鈕，可顯示陸地怪物尋路軌跡 3 秒，並以動態虛線動效提示怪物行進方向。
    *   跨局天賦系統提供持久進度，支援多級天賦升級，解鎖新砲台與提升基礎/屬性能力。
    *   砲台可進行「同系合成」或「異系配方合成」，產生更強的高階塔。

---

## 2. 核心遊戲循環 (Core Game Loop)

### 單局循環
```
主介面 → 開始遊戲 → 擺放砲台(築迷宮) → 開始波次 → 怪物尋路 → 砲台射擊 → 擊殺/扣血 → 波次結算 → 重複或失敗
```

### 跨局循環 (Meta-Progression)
```
單局結束 → 依存活波次獲得天賦點 → 天賦頁面分配天賦 → 下一局帶著強化的能力進入
```

## 4. 天賦系統 (Talent / Meta-Progression)

### 天賦點獲取
*   **失敗或通關**都會結算天賦點。
*   公式：`天賦點 = max(1, floor(存活波次 / 3))`
*   數據以 `localStorage` 持久化（儲存格式已重構為多級支援，舊存檔會自動轉換為新格式）。

## 5. 五行陰陽砲台體系

### 5.1 七種基礎砲台

| 屬性 | 名稱 | 花費 | 傷害 | 射程 | 特殊效果 |
|:---|:---|:---|:---|:---|:---|
| ⛰️ 土 | 岩壁塔 | 2g | 0 | 0 | 純牆壁，不攻擊（預設解鎖基礎建設） |
| 🔥 火 | 烈焰塔 | 12g | 20 | 4 | AOE 傷害（1格範圍 30%）（預設解鎖攻擊武器） |
| 💧 水 | 冰凍塔 | 10g | 8 | 5 | 減速 50%，90 幀（需天賦解鎖） |
| 🌿 木 | 纏繞塔 | 10g | 5 | 4 | DOT 3/幀，持續 60 幀 |
| ⚔️ 金 | 鏡刃塔 | 15g | 25 | 3 | 暴擊 20%（x2 傷害） |
| 🌑 陰 | 暗影塔 | 18g | 12 | 6 | 扣除 5% 最大 HP |
| ☀️ 陽 | 聖光塔 | 18g | 15 | 5 | 飛行怪 +50%，治療基地 0.1 |

### 5.2 同系合成 (Same-Element Merge)

兩座**同屬性 Lv1** 砲台 → 合成為 **Lv2 強化版**。效果全面提升。

### 5.3 異系配方合成 (Cross-Element Recipe)

兩座**不同屬性**砲台按五行相生關係合成：

| 配方 | 產物 | 效果 |
|:---|:---|:---|
| 木 + 火 (木生火) | 🌋 焚林塔 | 大範圍 AOE + DOT |
| 火 + 土 (火生土) | 🏔️ 熔岩塔 | 地面持續傷害區域 |
| 土 + 金 (土生金) | ⚒️ 鍛造塔 | 強化周圍 2 格友方塔傷害 +25% |
| 金 + 水 (金生水) | 🌊 寒鐵塔 | 高傷害 + 減速 60% |
| 水 + 木 (水生木) | 🌸 靈木塔 | 生成臨時障礙迫使怪物繞路 |
| 陰 + 陽 | ☯️ 太極塔 | 真實傷害 + 治療基地 |

### 5.4 五行相剋

```
金剋木 → 木剋土 → 土剋水 → 水剋火 → 火剋金
```
砲台攻擊相剋屬性怪物時傷害 **+30%**。

### 波次配置規則
*   每波根據波次號決定出場怪物的多樣性（隨波次增加）。
*   HP 隨波次倍率成長：`hpMult = 1.0 + (wave - 1) * 0.25`
*   每 3 波加入飛行怪，每 5 波出 Boss。
