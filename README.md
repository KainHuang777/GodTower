# 五行迷宮塔防 (Wuxing Maze Tower Defense)

> 以五行陰陽為核心的迷宮式塔防遊戲，玩家透過擺放砲台築起迷宮延長怪物路徑，並藉由跨局天賦系統持續強化能力。

---

## 🎮 遊戲簡介

**五行迷宮塔防**是一款執行於 Web 瀏覽器的純前端塔防遊戲，融合了：

- **迷宮建構**：砲台同時充當牆壁，玩家需在 80×40 網格中建立迷宮，迫使怪物走最長路徑（A\* 動態尋路）
- **多地圖與自訂編輯器**：內建 4 張不同格局與難度的關卡地圖（含折返走廊教學關卡），並內建獨立的 Canvas 地圖編輯器，供玩家自訂起點、終點、檢查點與地形障礙，並經 A\* 路徑驗證後存檔
- **五行陰陽系統**：7 種基礎屬性砲台（金木水火土 + 陰陽），含同系升級合成與異系配方合成
- **Meta 進度系統**：跨局天賦樹，支援多級升級，以 `localStorage` 持久化存儲
- **像素美術**：所有精靈以程式碼內建像素矩陣繪製，無需外部圖片資源

---

## 🕹️ 核心玩法

| 機制 | 說明 |
|:---|:---|
| **檢查點路徑** | 地圖設有 5 個必須按序訪問的檢查點，怪物需通過所有檢查點才能抵達基地 |
| **動態尋路** | 每次擺放或移除砲台後即時重算怪物路徑（A\*），確保路徑不被完全堵死 |
| **砲台合成** | 選取任意兩座砲台進行合成，同系升 Lv2，異系依五行相生關係產生特殊塔 |
| **路線預覽** | 戰鬥中可隨時查看陸地怪物的行走軌跡（動態虛線動效，顯示 3 秒後漸隱） |
| **五行相剋** | 金剋木 → 木剋土 → 土剋水 → 水剋火 → 火剋金；克制屬性傷害 **+30%** |

## 🏗️ 砲台體系

### 七種基礎砲台

| 屬性 | 砲台 | 費用 | 預設解鎖 | 特色 |
|:---:|:---|:---:|:---:|:---|
| ⛰️ 土 | 岩壁塔 | 2g | ✅ | 純牆壁，無攻擊力，最低成本阻路 |
| 🔥 火 | 烈焰塔 | 12g | ✅ | AOE 傷害（1 格範圍 30%） |
| 💧 水 | 冰凍塔 | 10g | 天賦 | 減速 50%，持續 90 幀 |
| 🌿 木 | 纏繞塔 | 10g | 天賦 | DOT 3/幀，持續 60 幀 |
| ⚔️ 金 | 鏡刃塔 | 15g | 天賦 | 暴擊 20%（x2 傷害） |
| 🌑 陰 | 暗影塔 | 18g | 天賦 | 扣除目標 5% 最大 HP |
| ☀️ 陽 | 聖光塔 | 18g | 天賦 | 飛行怪 +50% 傷害，治療基地 |

| 配方 | 產物 | 效果 |
|:---|:---|:---|
| 🌿 木 + 🔥 火 | 🌋 焚林塔 | 大範圍 AOE + DOT |
| 🔥 火 + ⛰️ 土 | 🏔️ 熔岩塔 | 地面持續傷害區域 |
| ⛰️ 土 + ⚔️ 金 | ⚒️ 鍛造塔 | 強化周圍 2 格友方塔傷害 +25% |
| ⚔️ 金 + 💧 水 | 🌊 寒鐵塔 | 高傷害 + 減速 60% |
| 💧 水 + 🌿 木 | 🌸 靈木塔 | 生成臨時障礙迫使怪物繞路 |
| 🌑 陰 + ☀️ 陽 | ☯️ 太極塔 | 真實傷害 + 治療基地 |

## 🎖️ 天賦系統 (Meta-Progression)

- **天賦點獲取**：`天賦點 = max(1, floor(存活波次 / 3))`，失敗與通關皆結算
- **多級升級**：所有天賦支援多次升級（最多 3～5 級），UI 顯示 `Lv.n/max`
- **持久化**：天賦資料儲存於 `localStorage`，重開遊戲不會重置
- **路線分支**：基礎強化 / 攻擊強化 / 五行解鎖 / 陰陽解鎖 四大路線

## 🚀 本地開發

```bash
# 安裝依賴
npm install

# 啟動開發伺服器
npm run dev

# 建置生產版本
npm run build

npm run preview
```

開發伺服器啟動後，開啟瀏覽器造訪 `http://localhost:5173`。
